Date

Format names can be mixed with various options. See below the most likely examples.

FormatDirectiveFilter
none
<fs-date fs-date="date"></fs-date> 		{{date | fsDate}}
{{'{{'}}date | fsDate{{'}\}'}}
{{format}}
<fs-date fs-date="date" fs-format="{{format}}"></fs-date> 		{{date | fsDate:format}}
{{'{{'}}date | fsDate:'{{format}}'{{'}\}'}}

Ago

There is only one format for ago, it includes both display format and a mouseover tooltip, as shown below

Note: the directive will update itself as time passes, the filter will not.

FormatDirectiveFilter
{{from_date | fsDate:'date-time'}}
<fs-date-ago fs-date="from_date"></fs-date-ago> 		{{from_date | fsDateAgo}}
{{'{{'}}from_date | fsDateAgo{{'}\}'}}
{{from_dates[10] | fsDate:'date-time'}}
<fs-date-ago fs-date="from_date" fs-format="date-time"></fs-date-ago> 		{{from_dates[10] | fsDateAgo:'date-time'}}
{{'{{'}}from_date | fsDateAgo:'date-time'{{'}\}'}}

Range

When dealing with 2 dates in a range, unnecessary repetition can be avoided using the “range” parameter. Note that range parameter works along with other date time format patterns.

formatContextDirectiveFilter
{{format}} {{range.desc}}
<fs-date-range fs-from="range.start" fs-to="range.end" fs-format="{{format}}"></fs-date-range> 		{{ [range.start,range.end] | fsDateRange:format}}
{{'{{'}} [range.start,range.end] | fsDateRange:'{{format}}'{{'}\}'}}

{{row.time}}

{{'{{'}}time | fsDateDuration:{{row.options}} {{'}\}'}} 		{{duration(row.time, row.options)}}
duration(time, {{row.options}})
